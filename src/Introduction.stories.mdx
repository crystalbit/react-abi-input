# React ABI Component Library

A modern React component library built with TypeScript, designed to be fully customizable and type-safe.

## Installation

```bash
npm install react-abi
# or
yarn add react-abi
```

## Usage

```jsx
import { Button } from 'react-abi';

function App() {
  return (
    <Button variant="primary" onClick={() => console.log('Button clicked!')}>
      Click me
    </Button>
  );
}
```
